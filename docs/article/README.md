# Artigo Cientifico

- Fonte LaTeX: `fraud_lakehouse_article.tex`
- PDF para leitura: `fraud_lakehouse_article.pdf`
- Renderizador de contingencia: `render_pdf.py`

O arquivo `.tex` usa a estrutura solicitada e pode receber diretamente o preambulo do template SBC. O PDF foi renderizado localmente a partir do mesmo texto para viabilizar a entrega em um ambiente sem distribuicao TeX instalada.
